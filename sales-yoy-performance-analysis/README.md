# Sales Year-over-Year Performance Analysis

## Business Context

This project examines departmental sales dynamics across two consecutive years.
The analysis aims to identify behavioral changes in purchasing patterns and their impact on business performance.

## Objective

* Compare revenue trends year-over-year
* Identify seasonal purchasing behaviors
* Evaluate performance by customer segment
* Support strategic commercial decision-making

## Analytical Approach

* Comparative time-series analysis
* Segment-level revenue evaluation
* Executive storytelling dashboard design

## Key Insights

* Seasonal effects influence departmental performance
* Customer profile shifts impact revenue distribution
* Growth opportunities identified in specific segments

## Tools

Power BI, DAX, Business KPI Design

## Notes

Dashboard layout designed to resemble executive presentation format.
