# Campaign Engagement Analytics

## Business Context

This project evaluates supporter engagement across advocacy campaigns by analyzing behavioral interaction and financial contribution patterns.

## Objective

* Measure campaign engagement effectiveness
* Analyze supporter participation trends
* Evaluate donation behavior by campaign
* Understand demographic engagement differences

## Analytical Approach

* Multi-metric engagement analysis
* Behavioral segmentation of supporters
* Campaign performance benchmarking

## Key Insights

* Engagement varies significantly by campaign theme
* Email interaction correlates with donation likelihood
* Demographic differences influence participation intensity

## Tools

Power BI, DAX, Segmentation Analysis

## Notes

Data structure designed to simulate real-world campaign analytics scenarios.
